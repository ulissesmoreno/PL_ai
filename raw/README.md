# raw/

Este diretório armazena documentos-fonte (PDFs, TXTs, relatórios, etc.) que serão ingestados na wiki.
This directory stores source documents (PDFs, TXTs, reports, etc.) to be ingested into the wiki.

**REGRA / RULE:** Os arquivos aqui são IMUTÁVEIS. Nunca edite documentos nesta pasta.
Files here are IMMUTABLE. Never edit documents in this folder.

## Como adicionar uma fonte / How to add a source

1. Coloque o arquivo aqui (ex.: `raw/relatorio-2026-01.pdf`).
2. Solicite ao Agente para "ingestar" o documento.
3. O Agente criará páginas em `wiki/` e atualizará `wiki/index.md` e `wiki/log.md`.
